What is machine learning?

It is a way to make sense of the data (which can be interpreted in different ways).

The trade-off

  • Bias vs. variance
  • Abstraction vs. specificity
  • Underfit vs. overfit

Linear Regression

Model

h = w_0 + w_1 x_1 + w_2 x_2 + ... + w_D x_D

  • Simple
  • Closed-form solution (D\downarrow)
